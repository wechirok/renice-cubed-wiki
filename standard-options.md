# Standard options

## Video settings

| Option | Description | Modpack |
|---|---|---|
| `graphicsPreset` | Graphics preset | `"custom"` |
| `renderDistance` | Render Distance: 10 Chunks | `10` |
| `simulationDistance` | Simulation Distance: 10 Chunks | `10` |
| `biomeBlendRadius` | Biome Blend: 1 block | `1` |
| `entityDistanceScaling` | Entity Render Distance: 75% | `0.75` |
| `entityShadows` | Entity Shadows: OFF | `false` |
| `cutoutLeaves` | See-Through Leaves: ON | `true` |
| `textureFiltering` | Texture Filtering: None | `0` |
| `mipmapLevels` | Mipmap Levels: 2 | `2` |
| `particles` | Particles: Decreased | `1` |
| `weatherRadius` | Weather Effect Radius: 5 blocks | `5` |
| `prioritizeChunkUpdates` | Chunk Builder Priority | `0` |

## Interface and game settings

| Option | Description | Modpack |
|---|---|---|
| `guiScale` | GUI Scale: 3 | `3` |
| `operatorItemsTab` | Operator Items Tab: ON | `true` |
| `tutorialStep` | Tutorial Hints: OFF | `none` |

## Hidden options

| Option | Description | Modpack |
|---|---|---|

| `onboardAccessibility` | Accessibility onboarding screen | `false` |
| `skipMultiplayerWarning` | Hides multiplayer warning screen | `true` |
| `telemetryOptInExtra` | Extra telemetry opt-in | `false` |
